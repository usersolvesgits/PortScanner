### Port Scanner (nmap-like) - Classe 4°

#### Tecnologie
- XAML - FluentUI theme pack - see: https://www.thomasclaudiushuber.com/2025/02/21/wpf-in-net-9-0-windows-11-theming/
- C# - WPF Project
- System.Net.Sockets

#### Required Feature
- Scansione porte su IP/hostname specifico
- Visualizzazione porte aperte/chiuse
- Gestione range porte da scansire
- Interfaccia grafica per inserimento parametri
- Visualizzazione risultati in tempo reale

#### Optional Feature
- Scansione UDP oltre TCP
- Service fingerprinting (identificazione servizio)
- Export risultati in JSON/CSV
- Salvataggio cronologia scansioni
- Multi-threading per scansioni veloci
- Timeout configurabile
- Scansione subnet intera
- Whois lookup per IP
- Rilevamento OS (OS fingerprinting)
- Ping sweep per discovery host attivi
- Scansione SYN stealth (half-open scan)
- Rilevamento versioni servizi (Service versioning)
- Filtri e ordinamento risultati
- Notifiche per porte critiche/note
- Statistiche scansione (durata, porte trovate, etc.)
